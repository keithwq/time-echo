# 题库改写提示词

## 任务
改写《时光回响》项目的访谈题库，使其更符合老年人的表达习惯和思维方式。

## 核心要求

### 1. 语言风格
- **朴实直接**：避免文艺腔、矫情的表达
- **口语化**：像拉家常一样，不要像写散文
- **具体实在**：问具体的事，不问抽象的感受
- **简短明了**：一句话说清楚，不绕弯子

### 2. 避免的表达方式
❌ "如果只用一种声音/味道来说..."（太文艺）
❌ "您心里是什么滋味？"（太抽象）
❌ "最像被点亮的感觉"（太矫情）
❌ "一种气息"、"一种氛围"（太虚）

### 3. 推荐的表达方式
✅ "您还记得...吗？"
✅ "当时是怎么回事？"
✅ "后来怎么样了？"
✅ "您当时怎么想的？"
✅ "那时候...是什么样的？"

### 4. 改写原则
1. **保留题目 ID 和类型**：不要改变
2. **保留核心意图**：问的主题不变
3. **改写问法**：用老年人习惯的方式问
4. **简化提示**：提示语也要朴实
5. **保留标签**：不要改变标签

## 示例

### 改写前（矫情）
```
题目 ID: 023
问题: 如果只用一种声音来说"这是童年"，您会想到什么？
提示: 先讲最像童年的那种声音。
```

### 改写后（朴实）
```
题目 ID: 023
问题: 您还记得小时候最常听到的是什么声音吗？
提示: 比如鸡叫、叫卖声、织布声。
```

---

### 改写前（抽象）
```
题目 ID: 019
问题: 离开学校那段时间，您心里是什么滋味？
提示: 先讲最深的一种感觉。
```

### 改写后（具体）
```
题目 ID: 019
问题: 离开学校那会儿，您当时是怎么想的？
提示: 是舍不得，还是想赶紧出去干活？
```

## 批量改写格式

请按以下格式输出改写结果：

```markdown

**问题**: [改写后的问题]
**提示**: [改写后的提示]

### 题目 2 (ID: 002)
**问题**: [改写后的问题]
**提示**: [改写后的提示]

...
```

## 注意事项
- 不要改变题目数量
- 不要改变题目 ID
- 不要改变题目类型（scene/emotion/people 等）
- 不要改变标签
- 只改写问题和提示语
