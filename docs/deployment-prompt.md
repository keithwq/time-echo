# 给 ChatGPT/Codex 的部署平台选择提示词

## 直接复制以下内容给 ChatGPT

---

我需要为一个公益回忆录项目选择合适的云部署平台，请帮我分析和推荐。

## 项目概况

**项目名称**：时光回响（Time Echo）
- 定位：适老化公益回忆录平台，帮助老年人记录人生故事
- 技术栈：Next.js 14 + Prisma + PostgreSQL + Tailwind CSS
- 用户群体：国内 40-70 岁中老年用户
- 预期流量：初期日活 100-500 人，后期可能到 1000-2000 人

## 技术需求

### 运行环境
- Node.js 18+
- Next.js 14（需要 SSR 支持）
- 需要 API 路由（/api/*）
- 支持静态页面生成

### 数据库
- PostgreSQL 14+
- 需要持久化存储
- 需要自动备份（每日）
- 数据量预估：初期 < 1GB，后期 < 10GB

### 存储需求
- 应用代码 + 依赖：约 500MB
- 数据库：初期 1GB，预留 10GB
- 用户上传图片：需要对象存储或文件系统
- 日志文件：约 1GB/月

### 性能需求
- CPU：1-2 核（最低），推荐 2 核
- 内存：2-4 GB（推荐 4GB）
- 带宽：5Mbps+（国内访问）
- 并发：支持 50-100 并发用户

### 网络需求
- 国内访问速度快（服务器位置）
- 需要自定义域名绑定
- 需要 HTTPS 证书
- 如果使用国内服务器，需要 ICP 备案

### 运维需求
- 支持 Git 自动部署（GitHub 集成）
- 环境变量配置
- 基础日志收集
- 监控告警（可选）

## 预算

- 月预算：50-150 元人民币
- 优先按量付费或包月
- 需要免费试用期（用于测试）

## 当前考虑的选项

1. **Sealos**（目前在用）
   - 价格：0.07元/小时 ≈ 50元/月
   - 优点：简单、容器化、按量付费
   - 缺点：相对小众、文档较少

2. **腾讯云轻量应用服务器**
   - 价格：约 60-80元/月
   - 优点：性价比高、国内速度快
   - 缺点：需要自行配置数据库

3. **阿里云 ECS + RDS**
   - 价格：约 120-180元/月
   - 优点：最稳定、生态完善
   - 缺点：价格较高、配置复杂

4. **Vercel + 国内 CDN**
   - 价格：免费-100元/月
   - 优点：Next.js 原生支持
   - 缺点：需要备案、数据库需另外解决

## 请帮我完成以下任务

1. **分析以上选项**，列出每个方案的优缺点
2. **推荐最适合的方案**，并说明理由
3. **提供详细的部署步骤**，包括：
   - 服务器选购配置
   - 环境安装（Node.js、PostgreSQL）
   - 项目部署流程
   - 域名绑定和 HTTPS 配置
   - 数据库备份设置
4. **提供成本优化建议**，如何在预算内获得最佳性能
5. **提供备选方案**，如果主方案不可行的替代选择

## 特别关注点

- **成本控制**：这是公益项目，需要极致省钱
- **数据安全**：老年人的人生回忆数据非常重要，不能丢失
- **访问速度**：中老年用户耐心有限，页面加载要快
- **维护简单**：团队技术能力有限，需要低维护成本

请给出详细的分析和建议。
