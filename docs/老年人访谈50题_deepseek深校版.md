# 时光回响·访谈题库（最终版）

本题库共50个主题，每个主题独立。老人勾选选项后，自动展开对应的下一级选项（无额外文字提示）。  
- **浅挖（1-2层）**：用于建立画像，不直接写入回忆录正文  
- **中挖（2-3层）**：可作背景或过渡  
- **深挖（3-5层）**：核心故事，直接写入回忆录  
- **短填空**：地名、人名、年份、数量等（一两个字即可）  

全程以点选为主，短填空为辅，不要求长篇打字。
