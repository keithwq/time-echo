## 📝 维护说明

### 如何添加新的 FAQ 项
编辑 `src/pages/help/faq.tsx` 中的 `faqData` 数组，添加新对象:
```typescript
{
  id: 16,
  question: "新问题标题",
  answer: "新问题答案"
}
```

### 如何修改使用指南步骤
编辑 `src/pages/help/guide.tsx` 中的 `steps` 数组，调整步骤内容。

### 如何修改隐私政策内容
编辑 `src/pages/help/privacy.tsx` 中的 `modules` 数组，更新相应模块内容。

### 表单反馈的后端集成
目前反馈表单使用 mock 提交（1.5 秒延迟后显示成功）。
若要集成真实后端，修改 `/contact/feedback.tsx` 中的 `handleSubmit` 函数，调用真实 API。
